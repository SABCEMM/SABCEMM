void AgentCross::preStepUpdate ( )

Draws initial thresholds and herdingPressure from respective uniform distributions (cf.

eq. 3.1, p.8), assigns a random initial decision. Check whether the agent must change position because price or herdingPressure are out of bounds and perform the position change, if necessary (cf p.6)
