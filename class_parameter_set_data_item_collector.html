vector< ParameterSetDataItemCollector > ParameterSetDataItemCollector::readFromXML ( tinyxml2::XMLElement *  dataItemCollectorClassesElement)
static

Read the content of the XML Block and generate valid ParameterSetDataItemCollector objetcs.

ParameterSet objects are not checked for consistency. This can be done via validate().

Note
All known parameters are read from the file. Additional/Unnecessary parameters should be ignored/go unused in the simulation.
Parameters
dataItemCollectorClassesElementXMLElementas parsed in XML from file
Returns
Vector of ParameterSetDataItemCollector objects. One for each defined DataItemCollector.

◆ validate()

bool ParameterSetDataItemCollector::validate ( )

Performs a consistency check on the ParameterSetDataItemCollector object.

It only checks if all necessary parameters are set, it is NOT checked weather the parameters make sens. This is done at object creation/initialisation mostly in the Simulation.cpp.

Returns
True if all necessary parameters are set, False if some parameters are missing.
