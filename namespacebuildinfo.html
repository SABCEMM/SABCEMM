Detailed Description

Holds variables that describe the machine that SABCEMM was built on.

Those variables are declared will be declared in Version.cpp which is generated from Version.cpp.in during build.

◆ GIT_VERSION

const std::string buildinfo::GIT_VERSION

A version string returned by git describe for the current commit.

For regular commits, this string will look like "<tag>-<#commits since tag>-hash". For releases, this string will be the name of the current tag. (like "v1.0").
