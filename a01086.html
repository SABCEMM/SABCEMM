void LatticeNeighbourhoodGenerator::generateNeighbourhood ( )
virtual

Arranges all agents in a lattice neighbourhood.

Agents are assigned based on their position in the vector of all agents. Each agent is assigned the agent above, below, left and right (geometrically) as neighbourhood. Agents on the edges are assigned agents from the corresponding edge on opposite site (i.e. there is a wrap around).
