Detailed Description

Container Class for the ExcessDemand.

Todo:
Klasse überarbeiten! Soll die Klasse globale Daten schützen? Dann muss excessDemand per Referenz gehalten werden, nicht per Wert.

Sind die Referenzen aus Performancegründen dort? Dann müssen die Referenzen entfernt werden, sie machen die Benutzung mühsam und geben gerade keinen Performancevorteil, da ein double ähnlich groß ist wie ein Pointer. Can be included in every class that needs access to the current excess demand. The excess demand can only be set by befriended implementations of the ExcessDemandCalculator.
